# Módulo de actividades  

En este modulo va todo lo referente a las actividades, a excepción de la conexión a la base de datos, ya que eso se hace en el módulo de la base de datos.  

## Contenido  
1. **screen**: La pantalla principal de las actividades, mostrando una lista de todas las que se crearon, además de permitir agregar más, editar, o borrar.
2. **view**: componentes extras que están en la *screen*.  
3. **model**: Abstracción de las funcionalidades de las actividades.
